# Checklist de Submissão para Lojas de Smart TV

## Samsung Seller Office (Tizen)

### Requisitos de Assets
- [ ] Ícone do app: 512x512 PNG
- [ ] Screenshots: 1920x1080 PNG (mínimo 4)
- [ ] Banner promocional: 1920x1080 PNG (opcional)
- [ ] Vídeo de demonstração: MP4 (opcional)

### Informações Obrigatórias
- [ ] Nome do app: RedFlix
- [ ] Descrição curta (80 caracteres)
- [ ] Descrição completa (4000 caracteres)
- [ ] Categoria: Entretenimento / Streaming
- [ ] Idiomas suportados: Português (Brasil)
- [ ] Países de distribuição
- [ ] Política de privacidade URL
- [ ] Termos de uso URL
- [ ] Email de suporte

### Requisitos Técnicos
- [ ] Pacote .wgt assinado com certificado de distribuição
- [ ] Testado em TV Samsung real
- [ ] Navegação por controle remoto funcional
- [ ] Player HLS funcional
- [ ] Sem erros de console

### Portal
URL: https://seller.samsungapps.com

## Notas Importantes

### Certificados e Assinaturas
- Samsung: Certificado de distribuição via Samsung Certificate Manager
- LG: Não requer certificado especial para submissão
- Google Play: Keystore próprio ou Google Play App Signing
- Amazon: Keystore próprio

### Tempo de Revisão Estimado
- Samsung: 5-10 dias úteis
- LG: 5-7 dias úteis
- Google Play: 1-3 dias úteis
- Amazon: 1-3 dias úteis

### Política de Privacidade
Deve incluir:
- Dados coletados
- Como os dados são usados
- Compartilhamento com terceiros
- Direitos do usuário
- Contato para questões de privacidade

### Classificação de Conteúdo
- Streaming de vídeo: Geralmente 12+ ou 16+
- Depende do conteúdo disponível na plataforma
- Pode variar por país

### Atualizações
- Manter versão consistente entre plataformas
- Documentar mudanças em cada atualização
- Testar em dispositivos reais antes de submeter
